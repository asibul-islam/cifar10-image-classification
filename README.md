# CIFAR-10 Image Classification (TensorFlow & PyTorch)

This project focuses on multi-class image classification using Convolutional Neural Networks (CNNs) and transfer learning techniques on the CIFAR-10 dataset.

## Dataset

CIFAR-10 is a benchmark dataset consisting of 60,000 32×32 color images across 10 classes:

- airplane
- automobile
- bird
- cat
- deer
- dog
- frog
- horse
- ship
- truck

## Objectives

- Build CNN models from scratch
- Implement models in both TensorFlow and PyTorch
- Improve performance using training techniques
- Evaluate models using classification metrics
- Analyze model errors and misclassifications
